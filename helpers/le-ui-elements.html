LE UI elements for Moodle

The following UI elements can be used to decorate your Moodle contents and create interactivity, all within the Moodle editor.

They use a combination of Bootstrap and FontAwesome. These are both 'baked in to Moodle' so it makes editing a breeze.

Forum Description - Each person posts one discussion

Post a discussion
  1. Select Add a new discusion topic
  2. Add a subject, use the activity title and your name e.g. Land use restrictions - John
  3. Add your contribution in the message section
  4. Select Post to forum.
Reply to a post
  1. Select the students' post from the Discussion list
  2. Select Reply
  3. Add comments
  4. Select Post to forum.

Risk Level and Actions Required

Risk Level

Action

Very High

Act immediately: STOP WORK. Steps must be taken to lower the risk level.

High

Act today: The proposed activity can only proceed, provided that:

  • Risk level has been reduced
  • Controls selected come from legislation, AS, CoP and/or operating procedures
  • Risk assessment approved
  • Safe Working Method prepared
  • Supervisor reviews effectiveness of controls

Medium

Act this week: The proposed activity can only proceed, provided that:

  • Risk level has been reduced
  • Risk assessment approved
  • Safe Working Method prepared

Low

Act this month: Manage by documented procedures.
